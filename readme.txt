# Fourier Image Reconstruction Visualizer

A Python application that reconstructs images using Fourier Transform and Epicycle Visualization.

## Features

- Extracts image contours using OpenCV
- Applies Fast Fourier Transform (FFT)
- Reconstructs images using animated epicycles
- Real-time visualization with PyQt6

## Technologies

- Python
- NumPy
- OpenCV
- PyQt6

## Run

```bash
pip install numpy opencv-python PyQt6
python src/main.py
